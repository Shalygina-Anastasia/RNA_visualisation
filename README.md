# RNA_visualisation
#  Визуализация вторичной структуры РНК (Dot-Bracket Notation)

> Учебный проект 1 курса по дисциплине «Структуры данных».  
> Реализован на Python с использованием `matplotlib` и `numpy`.

## О проекте
Программа принимает последовательность РНК и её описание в формате **dot-bracket нотации** (например, `...(((...)))...`), проверяет корректность записи и строит наглядную 2D-визуализацию вторичной структуры.  
В рамках проекта отработаны базовые алгоритмы работы со строками, проверка скобочных последовательностей, парсинг структурных блоков и построение графиков.
---
## Ключевые возможности
-  **Валидация нотации:** проверка баланса скобок `()`, допустимых символов и совпадения длин последовательности и нотации.
-  **Парсинг структуры:** автоматическое разбиение на блоки (свободные нуклеотиды, стебли и петли) с расчётом их геометрических параметров.
-  **Визуализация:** построение схемы с помощью `matplotlib`:
  - Последовательная линия цепи
  - Водородные связи (пунктирные линии)
  - Нуклеотиды в виде кружков с буквами
-  **Простой интерфейс:** удобный запуск в Jupyter Notebook или через терминал.
---
## Требования
- Python `3.8+`
- Библиотеки: `matplotlib`, `numpy`
